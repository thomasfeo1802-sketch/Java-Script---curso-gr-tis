JavaScript no HTML: Primeiros Passos no Navegador

Aviso Impostante: Esse curso não ensina HTML, essa aula vai apenas oque precisa para poder programar em java script na web

Até agora, você utilizou o JavaScript principalmente no console, focando em lógica e estrutura.
A partir desta aula, você começa a usar o JavaScript dentro do navegador, interagindo diretamente com páginas HTML.

Este é o primeiro passo para criar sites dinâmicos e interativos.

1. O que muda do console para o navegador?

No navegador:

O JavaScript controla elementos da página

O código reage a ações do usuário

O resultado aparece visualmente

O navegador executa o código automaticamente

O JavaScript continua sendo o mesmo, o ambiente é que muda.

2. Como o JavaScript funciona no HTML

O navegador entende JavaScript por meio da tag <script>.

Existem duas formas principais de usar JavaScript em uma página HTML:

Código direto no HTML

Código em arquivo externo .js (forma recomendada)

3. JavaScript dentro do HTML (não recomendado)

Exemplo simples:

<script>
    console.log("JavaScript funcionando no navegador");
</script>


Essa forma funciona, mas:

Mistura código com estrutura

Dificulta manutenção

Não é profissional

4. JavaScript em arquivo externo (boa prática)

A forma correta é separar:

HTML → estrutura

JavaScript → comportamento

Exemplo:

<script src="script.js"></script>


No arquivo script.js:

console.log("JavaScript carregado corretamente");


Essa separação deixa o projeto organizado.

5. Ordem de carregamento do script

A posição do <script> importa.

Boas práticas:

Colocar o <script> antes de fechar o </body>

Ou usar o atributo defer

Isso evita erros ao acessar elementos que ainda não carregaram.

6. Console do navegador

O navegador possui um console próprio:

Acesse com F12

Aba “Console”

Mostra console.log(), erros e avisos

Ele é sua principal ferramenta de depuração no front-end.

7. Testando se tudo está funcionando

Sempre teste:

Se o arquivo .js está sendo carregado

Se mensagens aparecem no console

Se não há erros

Um erro no console significa que algo precisa ser corrigido.

8. Objetivo desta aula

Ao final desta aula, você deve ser capaz de:

Criar um arquivo HTML simples

Conectar um arquivo JavaScript externo

Entender onde e como o JavaScript roda no navegador

Utilizar o console do browser para testes

Conclusão

A partir de agora, o JavaScript deixa de ser apenas lógica e passa a ser interação.
Você começou oficialmente a desenvolver para a web.
