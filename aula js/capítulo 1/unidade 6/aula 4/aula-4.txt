Manipulando Elementos do DOM com JavaScript (Prática)

Na aula anterior, você aprendeu a selecionar elementos.
Agora, você vai aprender a manipular esses elementos na prática.

Aqui o JavaScript começa, de fato, a alterar o que o usuário vê.

1. O que significa manipular um elemento?

Manipular um elemento é:

Alterar texto

Alterar valores

Alterar estilos

Mostrar ou esconder elementos

Mudar atributos

Tudo isso é feito exclusivamente com JavaScript.

2. Alterando texto de um elemento

Depois de selecionar um elemento, você pode mudar seu conteúdo.

elemento.innerText = "Novo texto";


Exemplo prático:

let titulo = document.getElementById("titulo");
titulo.innerText = "Texto alterado pelo JavaScript";

3. Alterando HTML interno

Quando você precisa inserir conteúdo com estrutura:

elemento.innerHTML = "<strong>Texto em negrito</strong>";


Use com cuidado, pois altera a estrutura interna do elemento.

4. Alterando valores (inputs)

Para elementos de entrada:

input.value = "Novo valor";


Isso é comum em formulários e sistemas interativos.

5. Alterando estilos com JavaScript

O JavaScript pode mudar estilos diretamente:

elemento.style.color = "red";
elemento.style.backgroundColor = "black";


Isso permite criar reações visuais baseadas em lógica.

6. Mostrar e esconder elementos

Um controle muito comum:

elemento.style.display = "none";   // esconde
elemento.style.display = "block";  // mostra


Isso cria estados visuais diferentes.

7. Alterando atributos

Você pode alterar atributos como:

src

href

disabled

title

Exemplo:

elemento.setAttribute("title", "Texto de ajuda");

8. Exemplo prático de manipulação
let mensagem = document.getElementById("mensagem");

mensagem.innerText = "Sistema ativo";
mensagem.style.color = "green";


Aqui você já controla:

Conteúdo

Aparência

Estado visual

9. Pensamento profissional

Sempre pense:

Qual é o estado atual?

O que deve mudar visualmente?

Quando essa mudança acontece?

Interface é reflexo da lógica.

10. Objetivo desta aula

Ao final desta aula, você deve ser capaz de:

Alterar texto e valores

Modificar estilos com JavaScript

Controlar visibilidade de elementos

Criar respostas visuais baseadas em lógica

Conclusão

Nesta aula, você saiu do controle interno e passou a modificar a interface com JavaScript.

Na próxima aula, você aprenderá a responder a ações do usuário, usando eventos, que é onde a interatividade realmente acontece.
