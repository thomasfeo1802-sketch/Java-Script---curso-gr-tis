A Unidade 6 marca um passo muito importante no curso:
a transição do JavaScript no console para o JavaScript rodando no navegador, interagindo diretamente com a página.

Aqui você começa a criar aplicações visuais e interativas.

Objetivo da Unidade 6

Ao final desta unidade, você será capaz de:

Conectar JavaScript a páginas HTML

Manipular elementos da página

Responder a ações do usuário

Criar sistemas simples com interface gráfica

Entender como sites dinâmicos funcionam

Conteúdos da Unidade 6
1. JavaScript no navegador

Diferença entre JavaScript no console e no browser

Onde colocar o JavaScript (<script>)

Arquivos externos .js

Ordem de carregamento

2. Introdução ao DOM (Document Object Model)

O que é o DOM

Estrutura da página como árvore

Elementos, nós e atributos

Como o JavaScript “enxerga” o HTML

3. Seleção de elementos

getElementById

getElementsByClassName

getElementsByTagName

querySelector e querySelectorAll

4. Manipulação de elementos

Alterar texto (innerText, innerHTML)

Alterar estilos (style)

Alterar atributos (src, href, value)

Criar e remover elementos

5. Eventos

O que são eventos

click, mouseover, keydown, etc.

addEventListener

Funções como resposta a ações do usuário

6. Estados no front-end

Controle de estado na interface

Mostrar e esconder elementos

Habilitar e desabilitar ações

Atualização visual baseada em lógica

7. Mini-projetos práticos

Contador interativo

Lista de tarefas (To-Do)

Sistema simples de cadastro

Jogo básico com interface

8. Boas práticas no front-end

Separação entre HTML, CSS e JavaScript

Código limpo e organizado

Evitar lógica no HTML

Reutilização de funções

Conclusão

A Unidade 6 é onde o JavaScript ganha vida.
Tudo o que antes acontecia apenas no console agora passa a:

Aparecer na tela

Reagir ao usuário

Criar experiências reais

Depois dessa unidade, você estará pronto para avançar para projetos completos, integração com CSS avançado e bases para frameworks no futuro.
