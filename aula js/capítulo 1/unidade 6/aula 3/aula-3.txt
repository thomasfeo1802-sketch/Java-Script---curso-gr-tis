Selecionando Elementos do DOM na Prática (JavaScript)

Nesta aula, o foco deixa de ser apenas conceito e passa a ser programação prática.
Você vai aprender como o JavaScript encontra elementos da página e como trabalhar com eles no código.

O HTML aqui continua sendo apenas suporte.
Tudo o que importa é o JavaScript.

1. Por que selecionar elementos?

Para o JavaScript fazer qualquer coisa na página, ele precisa:

Encontrar um elemento

Guardá-lo em uma variável

Manipular esse elemento

Sem seleção, não existe interação.

2. Selecionando elementos pelo id

O método mais simples e direto é:

document.getElementById("idDoElemento");


Exemplo prático:

let titulo = document.getElementById("titulo");

console.log(titulo);


Agora o JavaScript possui uma referência direta ao elemento.

3. Selecionando múltiplos elementos

Quando existem vários elementos do mesmo tipo, usamos métodos que retornam listas.

Por classe:
let itens = document.getElementsByClassName("item");

console.log(itens);


Isso retorna uma coleção de elementos.

Por tag:
let paragrafos = document.getElementsByTagName("p");

console.log(paragrafos);

4. Seleção moderna com querySelector

querySelector usa seletores parecidos com CSS e é muito usado na prática.

let botao = document.querySelector("#botao");


Seleciona o primeiro elemento que encontrar.

Para vários elementos:

let botoes = document.querySelectorAll(".botao");


Retorna uma lista.

5. Guardando elementos em variáveis

Sempre guarde elementos em variáveis:

let mensagem = document.getElementById("mensagem");


Isso permite:

Reutilizar

Manipular

Organizar o código

Nunca fique chamando document toda hora sem necessidade.

6. Exemplo prático completo
// Selecionando elementos
let titulo = document.getElementById("titulo");
let botao = document.querySelector("#botao");

// Exibindo no console
console.log(titulo);
console.log(botao);


Se o console mostrar os elementos, a seleção funcionou.

7. Lidando com listas de elementos

Quando você recebe vários elementos:

let itens = document.querySelectorAll(".item");


Você pode percorrer com loop:

for (let i = 0; i < itens.length; i++) {
    console.log(itens[i]);
}


Aqui você já une:

DOM

Arrays/coleções

Loops

8. Erro comum de iniciante

Se o elemento não existir:

O valor será null

Qualquer tentativa de uso gera erro

Por isso:

Garanta que o script carregue após a página

Sempre teste no console

9. Objetivo desta aula

Ao final desta aula, você deve ser capaz de:

Selecionar elementos do DOM

Guardar referências em variáveis

Trabalhar com listas de elementos

Usar loops com DOM

Testar tudo pelo console

Conclusão

Selecionar elementos é o primeiro passo da interatividade.
A partir daqui, você deixa de apenas observar a página e passa a controlá-la com JavaScript.

Na próxima aula, você vai manipular esses elementos, alterando texto, valores e estilos diretamente pelo código.
