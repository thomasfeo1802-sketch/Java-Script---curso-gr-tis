O DOM: Como o JavaScript Enxerga a Página

Nesta unidade, o foco não é ensinar HTML, e sim usar o mínimo de HTML necessário apenas para o JavaScript funcionar na web.

O HTML aqui é apenas um meio, não o objetivo.

1. O que é o DOM?

DOM significa Document Object Model.

O DOM é a forma como o navegador representa a página internamente, para que o JavaScript possa:

Ler informações

Modificar conteúdo

Reagir a eventos

Para o JavaScript, a página é um objeto, não um arquivo de texto.

2. Página como estrutura de objetos

Quando a página carrega, o navegador:

Lê o HTML

Constrói uma estrutura em forma de árvore

Disponibiliza essa estrutura ao JavaScript

Cada parte da página vira:

Um objeto

Com propriedades

E métodos

É isso que o JavaScript manipula.

3. Por que o DOM é importante?

Sem DOM:

O JavaScript não acessa a página

Não existe interatividade

Não existe front-end dinâmico

Com DOM:

O JavaScript controla o que aparece

Atualiza conteúdo

Reage ao usuário

O DOM é a ponte entre JavaScript e a web.

4. O objeto document

O principal acesso ao DOM é feito pelo objeto global document.

Ele representa:

O documento carregado

Toda a estrutura da página

Tudo começa a partir dele.

Exemplo conceitual:

document

5. O que o JavaScript pode fazer com o DOM

Usando o DOM, o JavaScript pode:

Acessar elementos

Ler textos e valores

Alterar conteúdo

Criar ou remover elementos

Controlar estados visuais

Tudo isso sem ensinar HTML, apenas usando o que já existe.

6. DOM não é HTML

Isso é muito importante:

HTML → linguagem de marcação

DOM → representação em memória

O JavaScript não altera o HTML original, ele altera o DOM que está carregado no navegador.

7. Pensamento profissional

Quando você escreve JavaScript para web, você pensa em:

Estados da interface

Ações do usuário

Reações do sistema

O DOM é apenas a ferramenta que permite isso.

8. Objetivo desta aula

Ao final desta aula, você deve ser capaz de:

Entender o que é o DOM

Saber o papel do document

Compreender como o JavaScript interage com a página

Diferenciar HTML de DOM

Conclusão

Você não está aprendendo HTML.
Você está aprendendo a usar JavaScript para controlar a web.
