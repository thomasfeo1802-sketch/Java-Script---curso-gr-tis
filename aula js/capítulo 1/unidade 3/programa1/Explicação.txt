Unidade 3 — Desafio 1

Funções e Lógica Matemática

Descrição do Desafio

Neste desafio, você irá criar um programa em JavaScript que calcula as raízes de uma equação do segundo grau utilizando a fórmula de Bhaskara.

Após calcular as duas raízes da equação, o programa deverá realizar operações matemáticas entre elas.

Requisitos do Programa

Criar uma função que receba os valores a, b e c

Calcular o delta

Verificar se o delta é válido (maior ou igual a zero)

Calcular:

Raiz A

Raiz B

Exibir no console os seguintes resultados:

Soma das raízes

Subtração entre as raízes (A − B e B − A)

Multiplicação das raízes

Divisão entre as raízes (A ÷ B e B ÷ A, quando possível)
