Introdução ao JavaScript

Vocês iniciaram o aprendizado da linguagem JavaScript.

Nesta aula, serão apresentados os conceitos de variáveis e os principais tipos de dados utilizados na linguagem.
As variáveis podem ser comparadas a caixas: elas servem para armazenar informações que poderão ser utilizadas ao longo do programa.

Em JavaScript, uma variável possui uma sintaxe específica para ser declarada:

let nome = valor;


Exemplo:

let vida = 100;


Também é possível declarar uma variável sem atribuir um valor inicial:

let nome;
let idade;
let vivo;
let altura;


Nesse caso, o valor da variável será undefined até que um valor seja atribuído.
Outra possibilidade é definir explicitamente o valor como null, indicando ausência intencional de valor.

Observação: Em JavaScript, o uso do ponto e vírgula (;) ao final de cada linha não é obrigatório, porém é considerado uma boa prática e altamente recomendado.

Tipos de Dados

As variáveis em JavaScript podem armazenar diferentes tipos de dados. Os principais são:

number — utilizado para valores numéricos;

string — utilizado para textos;

boolean — utilizado para valores lógicos (true ou false).

Esses tipos permitem representar informações essenciais e são amplamente utilizados no desenvolvimento de aplicações em JavaScript.
