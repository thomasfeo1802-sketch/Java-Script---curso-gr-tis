Operadores e Condicionais em JavaScript

Em JavaScript, existem diversos operadores utilizados para realizar cálculos, comparações e tomadas de decisão dentro do código. A seguir, são apresentados os principais tipos de operadores.

Operadores Aritméticos

Os operadores aritméticos são utilizados para realizar cálculos matemáticos:

+ — adição (soma)

- — subtração

* — multiplicação

/ — divisão

% — retorna o resto de uma divisão

Exemplo de uso:

let resultado = 10 % 3; // resultado será 1

Operadores Lógicos

Os operadores lógicos são utilizados para combinar ou inverter condições, sendo muito comuns em estruturas condicionais.

&& (AND) — retorna verdadeiro se todas as condições forem verdadeiras

if (a == 5 && b > 0) {
    // código executado se ambas as condições forem verdadeiras
}


|| (OR) — retorna verdadeiro se ao menos uma das condições for verdadeira

if (a == 0 || b == 0) {
    // código executado se uma das condições for verdadeira
}


! (NOT) — inverte o valor lógico da condição

if (a != 5) {
    // executa se "a" for diferente de 5
}

Operadores de Comparação

Os operadores de comparação são utilizados para comparar valores:

== — verifica se é igual

!= — verifica se é diferente

<= — menor ou igual

>= — maior ou igual

> — maior que

< — menor que

Operadores de Atribuição

Os operadores de atribuição servem para atribuir ou modificar valores de variáveis:

= — atribui um valor

let arroz = "comida";


+= — soma e atribui

let num = 5;
num += 1; // resultado: 6


-= — subtrai e atribui

num -= 1; // resultado: 4


*= — multiplica e atribui

num *= 2; // resultado: 10


/= — divide e atribui

num /= 2; // resultado: 5


%= — calcula o resto da divisão e atribui

num %= 2.5; // resultado: 0

Estruturas Condicionais

As condicionais são utilizadas para executar determinados blocos de código com base em uma condição.
Se a condição for verdadeira, um bloco será executado; caso contrário, outro bloco poderá ser executado.

Principais Estruturas

if — executa um bloco de código se a condição for verdadeira

else — executa um bloco de código caso contrário

Estrutura Básica de uma Condicional
if (condição) {
    // código executado se a condição for verdadeira
} else {
    // código executado se a condição for falsa
}

Exemplo Prático
let nome = "carlinhos";

if (nome == "carlinhos") {
    console.log("Nome correto");
} else {
    console.log("Nome incorreto");
}


Nesse exemplo, o código verifica o valor da variável nome e exibe uma mensagem diferente dependendo do resultado da comparação.
