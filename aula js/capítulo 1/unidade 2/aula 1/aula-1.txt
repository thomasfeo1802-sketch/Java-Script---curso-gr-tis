Introdução

Nesta aula, você aprenderá sobre as estruturas de repetição, também conhecidas como loops. Esses recursos permitem que um mesmo bloco de código seja executado várias vezes de forma automática, de acordo com uma condição definida.

Os loops são fundamentais na programação, pois evitam a repetição manual de código, tornam os programas mais eficientes e permitem lidar com grandes volumes de dados de maneira organizada.

O que são Loops?

Um loop é uma estrutura que executa um conjunto de instruções repetidamente enquanto uma condição for verdadeira ou até que um determinado critério seja atendido.

Em JavaScript, os principais loops são:

while

do...while

for

Cada um possui características próprias e é utilizado em situações específicas.

Loop while

O loop while executa um bloco de código enquanto uma condição for verdadeira.
Antes de cada repetição, a condição é verificada. Caso ela seja falsa logo no início, o código não será executado nenhuma vez.

Estrutura do while
while (condição) {
    // código executado enquanto a condição for verdadeira
}

Exemplo de uso
let contador = 0;

while (contador < 5) {
    console.log("Contador:", contador);
    contador++;
}


Neste exemplo, o código será executado enquanto o valor da variável contador for menor que 5.

Loop do...while

O loop do...while é semelhante ao while, porém possui uma diferença importante:
o bloco de código é executado pelo menos uma vez, mesmo que a condição seja falsa.

Estrutura do do...while
do {
    // código executado
} while (condição);

Exemplo de uso
let numero = 10;

do {
    console.log("Número:", numero);
    numero++;
} while (numero < 5);


Mesmo a condição sendo falsa desde o início, o código será executado uma vez.

Loop for

O loop for é geralmente utilizado quando se sabe exatamente quantas vezes o código deve ser repetido.
Ele reúne em uma única linha:

inicialização da variável de controle;

condição;

incremento ou decremento.

Estrutura do for
for (inicialização; condição; incremento) {
    // código executado a cada repetição
}

Exemplo de uso
for (let i = 0; i < 5; i++) {
    console.log("Repetição número:", i);
}


Neste caso, o loop será executado 5 vezes, com o valor de i variando de 0 a 4.

Comparação entre os Loops
Loop	Característica principal
while	Executa enquanto a condição for verdadeira
do...while	Executa pelo menos uma vez
for	Ideal quando o número de repetições é conhecido
Cuidados ao Utilizar Loops

Sempre garanta que a condição do loop possa se tornar falsa;

Evite loops infinitos, que podem travar o programa;

Utilize incrementos ou alterações de estado corretamente.

Considerações Finais

As estruturas de repetição são ferramentas essenciais para automatizar tarefas e tornar o código mais eficiente. O domínio dos loops while, do...while e for é indispensável para avançar em JavaScript e trabalhar com listas, jogos, sistemas e aplicações interativas.
