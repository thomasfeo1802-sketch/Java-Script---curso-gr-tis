Percorrendo Arrays com Estruturas de Repetição

Nesta aula, você aprenderá a trabalhar com arrays em conjunto com estruturas de repetição, aplicando de forma prática os conceitos estudados nas aulas anteriores. Até aqui, vimos o que são loops (while, do while e for) e conhecemos os principais métodos de arrays. Agora, chegou o momento de unir esses conhecimentos para manipular listas de dados de maneira automática e eficiente.

Essa é uma aula fundamental, pois grande parte das aplicações reais — como sistemas, jogos, sites e aplicativos — depende do processamento de listas de informações, como nomes, preços, pontuações, usuários e produtos.

Por que percorrer arrays é tão importante?

Um array pode armazenar vários valores, mas para trabalhar com esses dados de forma inteligente, é necessário acessar cada elemento individualmente. Fazer isso manualmente não é viável. É justamente aí que entram os loops.

Com estruturas de repetição, conseguimos:

Ler todos os valores de um array;

Aplicar regras e condições em cada elemento;

Calcular totais, médias e contagens;

Exibir informações dinamicamente;

Automatizar tarefas repetitivas.

Percorrendo arrays com for

O loop for é a forma mais tradicional de percorrer um array, pois trabalha diretamente com índices.

Nesse tipo de estrutura:

Começamos pelo índice 0;

Utilizamos a propriedade length para definir o limite;

Acessamos cada posição do array automaticamente.

Esse método é ideal quando precisamos saber a posição do elemento ou realizar operações baseadas no índice.

Percorrendo arrays com while

O while permite percorrer arrays quando precisamos de maior controle sobre a condição de repetição.

Ele é útil em situações onde:

A quantidade de repetições pode variar;

A condição de parada não depende apenas do tamanho do array;

O controle do índice precisa ser mais flexível.

Apesar de exigir mais atenção para evitar loops infinitos, o while reforça a compreensão da lógica de repetição.

Percorrendo arrays com for...of

O for...of oferece uma forma mais simples e legível de percorrer arrays.

Com ele:

Não precisamos trabalhar com índices;

A cada repetição, o valor do elemento já é acessado diretamente;

O código fica mais limpo e fácil de entender.

Esse tipo de loop é recomendado quando o foco está no conteúdo do array, e não na posição dos elementos.

Loops com condicionais

Durante a repetição, podemos usar estruturas condicionais (if, else) para tomar decisões com base nos valores do array.

Isso permite:

Filtrar dados;

Contar elementos que atendem a um critério;

Exibir mensagens diferentes conforme o valor;

Criar regras de negócio simples.

Essa combinação de loops, arrays e condicionais é a base de praticamente qualquer sistema.

Aplicações práticas

Ao final desta aula, você será capaz de:

Exibir todos os itens de um array automaticamente;

Somar valores numéricos armazenados em listas;

Verificar se determinado valor existe em um array;

Criar lógicas reais utilizando listas de dados.

Essas habilidades serão essenciais para os próximos conteúdos do curso, onde trabalharemos com estruturas ainda mais complexas e próximas do desenvolvimento profissional.
